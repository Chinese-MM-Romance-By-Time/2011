            痛得快要爆炸了，像有带刺的铁锤敲击着太阳穴。

            我知道 如今已无法弥补 

            闪光曳着尾巴 消失在后方 
            让我明白 已离你愈来愈远 
            空虚的窗里有着束手无策的我 
            假寐的午后 
            梦见 一如往常在我身畔微笑着的你 
            梦见 微笑的你对着凝视窗外的我 
            停下脚步 凝视微暗的天际良久 
            为何 偏在这种时候说不出口 
            as if in a dream…” 



            【四】 
            然而这并不是故事的最后。 
            “你妈X的黎鹤，你又尿床了你知不知道？！”一脸横肉的护士长冲上前去，毫不留情地两记耳光，黎鹤的脸立刻肿了起来，“你信不信，再有下次我他妈就把你的命根子剪掉！” 

            “哟，甭生气了，这死老头的情况您还不清楚么？”虎背熊腰的护工，正一脸厌恶地收拾着黎鹤的床铺，“操，恶心死了。护士长您不知道吧，这老变态前几天又到处给病友们分发他那同性恋故事的稿子，哟喂，把那几个新来的孩子吓得呀……” 

            “去你妈的！”彪悍的护士长一脚揣在黎鹤的胸口，“你丫真是恬不知耻！恶心的同性恋，欠操的贱货！”极度反同的护士长已经不知道撕烂了黎鹤的多少份手稿，“你还有多少？全他妈给我交出来！” 

            “呜，呜呜……”黎鹤蜷缩在病房的角落，任那些拳脚像暴雨一样落在他的身上，“廖翔会来救我的……呜呜……不要打我了……” 
            “廖翔？好嘛，你倒是挺能意淫的……”护工已经从黎鹤的枕头下翻出了他写的故事，语气充满嘲讽和不屑“你丫也老大不小了，这不，快六十六岁了吧，怎么还和个傻子一样呢？” 

            “哦不，是我说错了。你不是傻子，是疯子……哈哈哈……”护工毫不留情地把那几张发黄的破纸撕得粉碎，“黎鹤，我劝你以后老实点，少他妈动那些不三不四的脑筋，知道吗？”说着，把那把碎纸狠狠地散在黎鹤的脸上。 

            一直平静着的黎鹤忽然就不可自控地爆发了。 
            “啊！！！！啊啊啊啊啊啊！！！！！啊啊啊啊啊！！！！！”他抓起自己掉的差不多的头发，死命地往墙上撞去，“啊啊啊啊！！！！！啊啊啊啊啊！！！！”他痛苦地嘶吼着，嗓音沙哑可怕，不像是人发出的叫喊，倒更像是绝望的野兽。 

            “啊！！！！！！啊啊啊啊！！！！！！”凄厉的嘶喊在精神病院空旷的走廊回荡着，大厅里传来了别的病人的骚动声。 
            “他妈的，真烦，又要浪费一支镇定针了。”护士长翻了个白眼，从衣袋里掏出针筒，毫不手软地随意刺进黎鹤的脖子。 
            那个瘦骨嶙峋的老人眼球暴突，口水四溢，额头上已经满是鲜血。他张了张嘴，缓缓地倒在了地上。 
            健壮的护工像扛一头死猪一样把他扔回了还湿漉漉的床铺。 

            黎鹤，六十六岁，同性恋。被关进Dream精神疾病中心已经五十年。 
            他一辈子没有谈过恋爱。 
            他渴望爱情，他渴望生死相随。 
            在他的幻想中他是一个长相清秀的美少年，有一个很相爱的情人叫廖翔，他们苦恋十年，最终一起死在血泊中。 
            他想像的爱情故事是个悲剧，就如同他现实的生活一样。 
            可对于他来说，这个梦是华丽的、快乐的、美好的，是如同天堂一样光芒四射的存在。 
            然而在这里，他连做梦的权利都没有。 
            镇定剂逐渐发挥催眠作用，黎鹤平躺着，浑浊的双眼不可自已地流出眼泪，划过他沟壑一般的苍老面容，一滴一滴打湿了枕头。 
            如果我活在梦里就好了。 
            又倘若我能死在梦里就好了。 

            As if in a dream… 



            【后记】 
            这个文是之前睡前莫名想到的，必须说明一点的是，其实“梦和死亡”这个梗已经不新鲜了，我很喜欢的两部电影都是差不多的题材，一部是《Mulholland 
            Drive》，一部是《Stay》，都是非常发人深思的文艺片。当然构思什么的比我高端多了，那才真是一环套一环啊……我想说我的虐点其实不在第三章的结尾，而是在第四章。一切都是黎鹤的幻梦啊有木有……黎鹤真的很苦逼啊有木有……还有就是《As 
            if in a dream》是我的本命彩虹乐队的歌^_^ 
            然后我来说说文里几个细节： 
            黎鹤16岁被关进精神病院，而他的幻想中，他16岁遇上了廖翔。 
            精神病院叫Dream，黎鹤幻想中的咖啡馆也叫Dream。 
            黎鹤喜欢写作，他的幻想中，黎鹤是一名颇受欢迎的作家。 
            黎鹤在精神病院受尽虐待，而在他的幻想中转化为情人对他的家庭暴力。 
            黎鹤在精神病院经常被镇定针刺入脖子，他的幻想中，他死于脖颈上的刀伤。 
            文章开始前大段的描述如“晕眩”“头痛”“胸闷”等都是黎鹤死亡前的感觉，而在大街上的飘忽感是因为这是一个梦，类似于灵魂出窍的感觉吧。 
            黎鹤把自身的精神疾病转嫁到了幻想中的廖翔身上。
